## ⚠️ 平台兼容性说明

本迁移包**仅面向 OpenHanako / HanaAgent** 用户。

如果你使用的是 **Codex、Claude Code、Cursor、Cline** 等其他 MCP 客户端，请注意以下几点不通用：

| 环节 | 本包的做法（仅 Hana 适用） | 其他平台的做法 |
|------|--------------------------|---------------|
| MCP 配置文件 | `%USERPROFILE%\.hanako\plugin-data\mcp\config.json`，含 Hana 专有字段 | 项目下的 `.mcp.json` 或 `claude.json`，标准 MCP 格式 |
| 配置方式 | 手动编辑 JSON 或复制示例文件 | 对话让 AI 自行配置，或按各平台文档操作 |
| connector ID | 必须与 Hana 的 agent config.yaml 对齐 | 无此限制 |

本包中的 `hana-config-example/` 目录下的配置文件、以及所有指引中涉及手动编辑 `.hanako` 目录的操作，在其他平台上均不适用。

如需在其他平台使用 `origin-mcp`，建议参考：
- https://github.com/Ge-Shun/origin-mcp   （官方 README 附各平台配置示例）
